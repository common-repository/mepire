== Description ==
=米派是什么 =
米派(www.mepire.com)是一款国内首创、创新型的交互式图片工具栏。网站主开启该插件后，即可将网站静态图片转变成具有丰富互动功能的交互式图片。米派工具栏的引入，可以帮助网站主轻松地将“社会化”，“电子商务”等现代互联网的核心元素和网站的静态图片无缝结合，全面提升网站的粘性及流量和商业价值。

=米派能做什么 =
米派(www.mepire.com)是一个简单而强大的互动图片工具栏。米派工具栏涵盖了一键式的社会化图片分享功能，图片内电子商务和广告等功能，这些功能不仅大大提升了网站的粘性和流量，也为网站提供了全新的盈利模式。 

=公司简介 =
武汉昊奇科技有限公司是一家致力于提供交互式图片与交互式视频服务的创新型科技公司。公司凭借在图像处理，视频跟踪，以互联网领域深厚的技术背景，国内首创地推出了互动图片工具“米派图片工具栏”以及互动视频技术服务平台“好奇看点”(www.videoclick.cn)两大创新产品。


== Installation ==
1. 将下载后的压缩包解压，得到一个名为mepire文件夹（包括一个.php文件、一个.txt文件以及五张.jpg图片），将该文件夹上传到wordpress的 '/wp-content/plugins/'目录下，然后在WordPress后台插件菜单下激活。

2. Mepire插件启用后，进入设置页面按照设置向导获取网站code和uid填入表格提交即可。


== Frequently Asked Questions ==
=米派是什么 =
米派(www.mepire.com)是一款国内首创、创新型的交互式图片工具栏。网站主开启该插件后，即可将网站静态图片转变成具有丰富互动功能的交互式图片。米派工具栏的引入，可以帮助网站主轻松地将“社会化”，“电子商务”等现代互联网的核心元素和网站的静态图片无缝结合，全面提升网站的粘性及流量和商业价值。

=米派能做什么 =
米派(www.mepire.com)是一个简单而强大的互动图片工具栏。米派工具栏涵盖了一键式的社会化图片分享功能，图片内电子商务和广告等功能，这些功能不仅大大提升了网站的粘性和流量，也为网站提供了全新的盈利模式。 

=我必须要在Mepire网站注册后才能正常使用吗 =
是的，用户注册会员的目的在于获取贵网址对应的 code和uid，一个网址对应一组code和uid，只有网址与code和uid匹配成功插件才能正常启用。

=获取code和uid是免费的吗=
是的，网站用户注册、获取code和uid都是免费。

=用户如何使用米派=
用户在对于启用了米派插件的网站时，会发现网站中的图片左下角有米派的标识图案， 用户鼠标滑过网站中的图片，可看到米派工具栏及与图片相关的信息。 用户只需要点击工具栏上的功能按钮，既可以在图片内完成如图片分享，留言互动，网上购物等动作。 要隐藏米派工具栏，仅需鼠标离开图片区域即可。

=为什么我启用插件后图片上仍然没有出现mepire工具栏=
1. 您的图片可能过小，不符合加载mepire工具栏的条件，mepire默认对250*250以上图片加载插件（如果参数不符，请访问mepire官网查看细则）。
2. 检查您的code和uid是否设置正确，查看填写域名是否与网址相符，只有网址与code和uid匹配成功插件才能正常启用。
